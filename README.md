# Porsche-Car-Showroom
A Python-based desktop application built using Tkinter and MySQL to simulate a car showroom billing and customer management system.

## Features
- Display car models with images
- Fetch car details from database
- Select car using image buttons
- Enter customer details
- Generate a billing summary
- Input validation and error handling

## Tech Stack
- Python
- Tkinter
- MySQL
- Pillow (PIL)

## Project Structure
Porsche-Car-Showroom/
├── main.py
├── database.sql
├── README.md
└── images/


## Setup Instructions

### Install dependencies


pip install mysql-connector-python pillow


### Database setup


source database.sql


### Run the application


python main.py
